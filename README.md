# 🚀 Student Management System – Flask Web App

## 📌 Overview
A secure and scalable web-based Student Management System built using Flask. The application allows authenticated users to manage student records efficiently with complete CRUD functionality and database integration.

---

## 🔐 Key Features
- User Registration & Login Authentication
- Secure Password Hashing using Flask-Bcrypt
- Session Management using Flask-Login
- Add / Update / Delete Student Records
- View and Search Student Data
- Database Integration using SQLAlchemy
- Responsive UI Design

---

## 🛠 Tech Stack
- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-Bcrypt
- HTML, CSS, Bootstrap
- SQLite / MySQL

---

## ⚙ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/sanidhyabhandariofficial/student-management-system-flask.git

### 2️⃣ Navigate to project folder
cd student-management-system-flask

### 3️⃣ Install dependencies
pip install -r requirements.txt

### 4️⃣ Run the application
python app.py

App will run at:
http://127.0.0.1:5000/
